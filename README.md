запуск прокта:

1) запустить apache и SQL в XAMPP
база данных будет находитcя по адресу 
http://localhost/phpmyadmin/

логин: root
пароля нет

при первом запуске надо создать базу данных с названием 
`travel_agency_db`
---

2) запуск бэкенда
надо открыть VS Code в папке demo
нажать на выпадающее меню из стрелочки в правом верхнем углу
нажать "Run Java"
---

3) запуск фронтенда

надо открыть терминал в папке frontend и написать 
`npm run dev`

фронтенд доступен по адресу
http://localhost:5173/

логин: agent@mail.com
пароль: admin

аккаунт пользователя

пароль: 123
